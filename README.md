# comandos_git

### Bash Keyboard Shortcuts

- Ctrl + A - Move o cursor para o inicio da linha
- Ctrl + E - Move o cursor para o final da linha
- Ctrl + K - Apaga o restante da linha a partir da posição do cursor
- Ctrl + W - Apaga a última palavra
- Ctrl + L - Limpa toda a tela. Equivalente ao comando clear
